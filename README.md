# AI Course Portfolio - Computer Assignments

## 📚 Course Overview
This repository contains a comprehensive collection of five computer assignments from the Artificial Intelligence course, covering a wide spectrum of AI concepts from classical search algorithms to modern deep learning and natural language processing. Each assignment demonstrates the implementation of fundamental AI algorithms from scratch, providing hands-on experience with both theoretical concepts and practical applications.

## 📋 Assignments Summary

| Assignment | Topic | Key Techniques |
|------------|-------|----------------|
| **CA1** | Search Algorithms | BFS, IDS, A*, Weighted A* for Lights Out Puzzle |
| **CA2** | Genetic Algorithm & Game AI | Genetic Algorithm, Minimax with Alpha-Beta Pruning |
| **CA3** | Ensemble Learning & Boosting | AdaBoost, Bagging, Random Forest, Decision Trees |
| **CA4** | Deep Learning from Scratch | Neural Networks, Backpropagation, CNNs | 
| **CA5** | NLP & Clustering | Text Preprocessing, K-Means, DBSCAN, Hierarchical Clustering |
